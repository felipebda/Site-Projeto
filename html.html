Estrutura básica de uma página HTML

A estrutura básica de uma página HTML, podemos ver as principais tags necessárias para que o documento seja corretamente interpretado pelos browsers(navegadores).


Titulos

Títulos são definidos com as tags <h1> a <h6>. A <h1> define o título maior. A <h6> define o título menor.

<h1>Este é um título</h1>

<h2>Este é um título</h2>

<h3>Este é um título</h3>

<h4>Este é um título</h4>

<h5>Este é um título</h5>
<h6>Este é um título</h6>

Parágrafos

O HTML automaticamente adiciona uma linha em branco extra antes e após um título. O HTML adiciona automaticamente uma linha em branco extra antes e após um parágrafo.

<p>Este é um parágrafo</p>

<p>Este é outro parágrafo</p>
